# Administrator Manual

Welcome to the Admin Control Panel. As an Administrator, you have elevated privileges to manage the entire Statsang Portal.

## 1. Admin Dashboard
A global overview of portal metrics.
- Track total active users, global event counts, and recent notice publications.

## 2. User Management
Full CRUD (Create, Read, Update, Delete) control over the community.
- **Add Users**: Create new accounts and assign them auto-generated User IDs (e.g. SAT1001).
- **Edit Profiles**: Update usernames and emails.
- **Toggle Status**: Suspend accounts by toggling their status to `INACTIVE`. Inactive users cannot log in.

## 3. Event Management
Coordinate community gatherings.
- Publish, edit, and delete upcoming Events.
- **Event Registrations**: View a global table of every user registered for every event to assist with capacity planning.

## 4. Notice Board
Broadcast messages to the community.
- Create announcements and flag them as `LOW`, `MEDIUM`, or `HIGH` priority.

## 5. Attendance (Bulk Actions)
Log attendance rapidly for the entire community.
- Select a date and session name.
- Check off users who were present from a paginated table.
- Click **Mark Present** or **Mark Absent** to apply changes in bulk.

## 6. Global Journals
Monitor community engagement.
- View a paginated list of all journal entries written by all users. This view is read-only.

## 7. Gallery Management
Curate the visual history of the community.
- Upload images directly to Cloudinary.
- Manage and delete existing photos.

## 8. Reports
Generate analytics.
- View overall attendance ratios and identify top attendees within the community.

## 9. Account Settings
Manage your own Admin profile.
- Update your admin username and email.
- Change your secure password.
