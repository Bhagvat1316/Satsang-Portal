# 🛕 Satsang Portal

A full-stack Community Management Portal developed for managing Satsang activities including user management, attendance tracking, journals, notices, events, gallery management, and reporting.

---

# Features

## User

- Secure Login
- Dashboard
- Attendance History
- Event Registration
- Personal Learning Journal
- Notices
- Gallery
- Profile Management
- Change Password

## Admin

- Dashboard
- User Management
- Attendance Management
- Journal Management
- Event Management
- Event Registrations
- Notices Management
- Gallery Management
- Reports
- Account Settings

# Installation

## Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/SATSANG_PORTAL.git
```

## Backend

```bash
cd Satsang_Portal_Backend

npm install

npm run dev
```

## Frontend

```bash
cd Satsang_Portal_Frontend

npm install

npm run dev
```

---

# Environment Variables

Create a `.env` file inside the backend folder.

```
DATABASE_URL=your_database_url

PORT=5000

JWT_SECRET=your_jwt_secret

CLOUDINARY_CLOUD_NAME=your_cloud_name

CLOUDINARY_API_KEY=your_api_key

CLOUDINARY_API_SECRET=your_api_secret
```
